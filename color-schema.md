# Website Color Pattern (Wedding Scrapbook)

Strong opinion: your palette is great. The key is to use **one main accent** and keep the rest as **supporting or neutral** so the site looks intentional and not chaotic.

## Color roles

| Role | Hex | Use it for |
|---|---|---|
| Background base | `#fde4ea` | Page background, large soft areas |
| Primary accent | `#813d6e` | Headings, primary buttons, links, key icons |
| Secondary accent | `#d7adcd` | Hover states, badges, highlights, decorative borders |
| Card / paper | `#c2cbec` | Card backgrounds, section blocks |
| Muted neutral | `#b5c1b5` | Subtle borders, dividers, muted UI text |
| Olive accent | `#a6a261` | Small highlights, tags, tiny icon accents (use sparingly) |
| Deep olive | `#7d7648` | Footer accents, darker text, hover accents |

## Button and link recipe

| Element | Colors |
|---|---|
| Primary button | Background `#813d6e`, text `#fde4ea`, hover background `#7d7648` |
| Secondary button | Background `#d7adcd`, text `#813d6e`, hover background `#c2cbec` |
| Links | Color `#813d6e`, hover underline plus `#7d7648` |

## Simple rules to keep it looking designed

- Use `#813d6e` as the only strong CTA color.
- Use olives (`#a6a261`, `#7d7648`) as seasoning only, about 5 to 10 percent of the UI.
- Keep big areas calm: mostly `#fde4ea` with cards in `#c2cbec`.
